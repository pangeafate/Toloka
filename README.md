# Toloka — Food Delivery RL Environment

Interactive pitch deck for the 30-minute project scoping call. React + Vite SPA, arrow-key navigation, light-theme Toloka brand styling.

**Live URL:** https://pangeafate.github.io/Toloka/

## Dev

```bash
npm install
npm run dev        # http://localhost:5173/Toloka/
npm run build
npm run lint
```

## Deploy

Push to `main`. GitHub Actions builds and publishes to `gh-pages` automatically. Total deploy time: ~40–60s.
